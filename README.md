# Proyek UAS: Tracker Nabung Target (Saving Goals)

Proyek ini adalah aplikasi web untuk melacak target menabung (Saving Goals) dengan visualisasi progres, pencatatan setoran (deposit), estimasi waktu selesai, dan statistik tabungan.

## Struktur Proyek

Proyek ini menggunakan pemisahan Frontend dan Backend secara terpisah:

- **`client/`**: Frontend menggunakan React.js yang dibangun dengan Vite dan Vanilla CSS.
- **`server/`**: Backend menggunakan Node.js, Express.js, Prisma ORM, dan MySQL (saat ini sebagai placeholder).
